## 5. 주요 기능
> **1. 선택한 지역의 전기차 충전소 목록 및 상세정보**

- 선택한 지역의 충전소 목록 제공 및 지도에 충전소 위치 표시
    - 지도 위 또는 지도 주변에 그래프 기반 요약 정보 제공
    - 충전소 클릭 또는 목록에서 선택 시 해당 충전소의 상세 정보 표시

> **2. 실시간 사용 가능 충전기 조회**

- 실시간 전기차 충전기의 사용 가능 여부 조회
- 리스트 뷰 또는 그래프 뷰를 주요 선택 화면으로 제공
    - 리스트 항목 또는 그래프 요소 선택 시, 현재 사용 가능한 충전기 목록 표시
 
> **3. 전기차 관련 커뮤니티 검색 기능**
  
- 전기차 관련 커뮤니티 사이트([이브이포스트](https://www.evpost.co.kr/wp/))에 작성된 게시글 검색 기능
- 검색 결과를 DB에 저장 및 제목과 하이퍼링크를 리스트 뷰로 제공
