# 사용 데이터

현재 csv및 api 준비 완료

## CSV 데이터

전국 전기차 충전소 사용량 데이터

주요 컬럼

* 광역지자체
* 시군구
* 사용량(키로와트시)
* 충전용량별 사용량

활용

* 지역별 충전소 사용량 분석
* 충전소 위치 시각화
* 충전용량 필터링

---

## Open API 데이터

한국환경공단 전기차 충전소 API

제공 정보

* 충전소명
* 주소
* 위도 / 경도
* 충전용량
* 충전방식
* 충전기 상태
* 운영기관

---

# 전기차 

## 1. 실시간 사용 가능 충전기 조회

선택한 지역의 사용 가능한 충전기를 실시간 조회

### 제공 정보

* 충전소명
* 위치
* 충전용량
* 충전방식
* 이용시간

---

## 2. 최적 충전소 추천

충전 가능 여부와 충전 성능을 기반으로 최적의 충전소 추천

### 추천 기준

* 사용 가능 여부
* 충전용량
* 사용자와의 거리
* 충전기 상태

---

## 3. 충전소 혼잡도 분석

충전소별 사용 가능한 충전기 비율을 계산하여 혼잡도 분석

### 혼잡도 구분

* 원활
* 보통
* 혼잡

### 기대 효과

* 대기시간 감소
* 충전소 이용 효율 향상

---

## 4. 충전기 고장률 분석

충전기 상태 데이터를 활용하여 고장률 분석

### 분석 항목

* 지역별 고장률
* 충전소별 고장률
* 운영기관별 고장률

### 활용 방안

* 충전소 신뢰도 평가
* 최적 충전소 추천 점수 반영

---

- 기사 URL

**분석 항목**

- 지역별 화재 건수
- 연도별 화재 발생 추이
- 충전중 / 주차중 / 주행중 분류
